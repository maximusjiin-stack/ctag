# CTAG LIFE Next.js

바로 실행 가능한 Next.js 웹사이트 프로젝트야.

## 실행

```bash
npm install
npm run dev
```

그다음 브라우저에서:

```bash
http://localhost:3000
```

## 포함된 파일
- `app/page.tsx`
- `app/layout.tsx`
- `app/globals.css`
- `tailwind.config.js`
- `postcss.config.js`
- `tsconfig.json`

## 특징
- 랜딩 섹션 포함
- 모바일 앱 프레임 데모 포함
- 홈 / 분석 / 스캔 / 추천 / 리워드 탭 전환
- 추천/리워드 검색 필터링
- 스캔 더미 애니메이션
